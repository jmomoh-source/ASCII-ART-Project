# SECURITY POLICY
## Supported Versions
At any given time, only the `main` branch of this repository is supported for security updates.

## Reporting a Vulnerability
We take the security of this project extremely seriously. If you discover a security vulnerability, please do NOT report it by publicly creating a repository issue!

Instead, **please send an email to Jezreelglobal@gmail.com** with the details of the vulnerability.

Please provide:
- A description of the vulnerability.
- Clear steps to reproduce the vulnerability.
- Any potentially affected platforms or environments.

We will acknowledge receipt of your vulnerability report within 72 hours, and we will strive to resolve the issue and publish a fix as soon as possible.
